# APLIKASI CHATBOT UNTUK PENDIDIKAN
Aplikaski ini dapat membantu siswa, pengajar, dan tenaga pendidik. Aplikasi ini mirip dengan ChatGPT tetapi lebih simple dan mudah untuk dimengerti oleh semua kalangan.
